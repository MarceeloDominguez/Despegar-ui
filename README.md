## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/Despegar-ui.git
```

Go to the project directory

```bash
  cd Despegar-ui
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
